# jguni-project

A modern web application built using Next.js.

## Features

* Responsive UI
* Fast performance
* Modern frontend design

## Technologies Used

* Next.js
* React
* CSS

## How to Run

1. Install dependencies

```bash
npm install
```

2. Start development server

```bash
npm run dev
```

3. Open in browser

```text
http://localhost:3000
```
